An intro to my research:

One problem we worked on is showing a problem in Knot Theory to be in the computational complexity class NP. Given two connected prime alternating reduced diagrams of a link L, we aim to understand the complexity of determining whether they represent the same link. From the work of Kauffman,Thistlethwaite, and Murasugi, we know that reduced alternating diagrams have minimal crossing numbers. Additionally, the Tait Flyping Conjecture, proven by Menasco and Thistlethwaite, implies that any two such diagrams of the same link can be related through a sequence of flype moves. We determine how many flype moves are necessary to transform one diagram into another. We provide a polynomial upper bound on the number of flypes in terms of the crossing number of the diagrams.

Note: This result is true for alternating diagrams only. For non-alternating links, the prime reduced diagram does not necessarily have minimal crossings. For example, Goeritz Unknot diagram is prime and reduced but does not have minimal crossings.

Fun problem: We know that Reidemeister moves increase the number of crossings on Goeritz Unknot. However, we can use flype moves on Goeritz Unknot to show that it is an Unknot. How many flype moves will be required to get the unknot diagram from Goeritz Unknot?

One of our projects focused on computing the meridian length for an infinite family of links that are closures of 4-braids. While software like SnapPy makes it possible to compute link invariants, it actually provides approximate values. In contrast, using the Thistlethwaite and Tsvietkova methods, I derived an exact algebraic expression for the meridian length of the infinite family. Moreover, SnapPy is not well-suited for handling the computation of hyperbolic invariants for infinite families of links, as the time complexity grows exponentially as the number of crossings increases, making it impractical. In our work, we developed a generalized formula that provides an exact value of meridian length for the infinite family.
